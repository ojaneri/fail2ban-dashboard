## Português

### 📝 Sobre o Projeto
O **Fail2Ban SOC Dashboard** é uma ferramenta de monitoramento e visualização em tempo real para o serviço Fail2Ban. Ele transforma logs de segurança complexos em um painel interativo (SOC style), permitindo identificar padrões de ataque, origens geográficas e gerenciar banimentos de forma intuitiva.

### 🚀 Stack Tecnológica
- **Backend:** FastAPI (Python 3.12+)
- **Banco de Dados:** SQLite (com suporte a SQLAlchemy)
- **Frontend:** React (Interface moderna e responsiva)
- **Visualização:** 
  - **Leaflet:** Mapa mundial interativo com heatmap.
  - **Chart.js:** Gráficos de pizza, barras e linhas para estatísticas de jails e ataques.
- **Gerenciamento de Pacotes:** `uv` (Astral)

### ⚙️ Configurações do Fail2Ban
Para que o dashboard funcione corretamente, ele monitora os logs padrão em `/var/log/fail2ban.log`.
As jails recomendadas incluem:
- `sshd`: Proteção contra ataques de força bruta em SSH.
- `nginx-http-auth`: Proteção para áreas restritas do servidor web.
- `recidive`: Banimentos de longo prazo para reincidentes.

## English

### 📝 About the Project
The **Fail2Ban SOC Dashboard** is a real-time monitoring and visualization tool for the Fail2Ban service. It transforms complex security logs into an interactive dashboard (SOC style), allowing you to identify attack patterns, geographic origins, and manage bans intuitively.

### 🚀 Tech Stack
- **Backend:** FastAPI (Python 3.12+)
- **Database:** SQLite (with SQLAlchemy support)
- **Frontend:** React (Modern and responsive interface)
- **Visualization:** 
  - **Leaflet:** Interactive world map with heatmap support.
  - **Chart.js:** Pie, bar, and line charts for jail and attack statistics.
- **Package Management:** `uv` (Astral)

### ⚙️ Fail2Ban Configuration
For the dashboard to function correctly, it monitors the standard logs at `/var/log/fail2ban.log`.
Recommended jails include:
- `sshd`: Protection against SSH brute force attacks.
- `nginx-http-auth`: Protection for restricted web server areas.
- `recidive`: Long-term bans for repeat offenders.
